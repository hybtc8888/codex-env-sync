# Using Superpowers
- Check the skill index before acting.
- Load only skills that clearly match the current user request.
- User, AGENTS.md, and project instructions take priority.
- Read a skill body only after selecting that skill.
- Prefer the `.codex` copy when duplicate skills exist.
- Load frontend, document, spreadsheet, presentation, or browser skills only for matching tasks.
- If no skill applies, proceed normally with Codex tools.
